## 🚨 Troubleshooting Guide

### Training Issues

| Symptom | Likely Cause | Solution |
|---------|--------------|----------|
| **Loss = NaN** | Learning rate too high | Reduce LR by 10x |
| **Loss plateaus early** | Learning rate too low | Increase LR by 2-5x |
| **Overfitting (train << val)** | Model too complex | Add dropout, reduce layers |
| **Underfitting (train ~= val, both high)** | Model too simple | Add layers, reduce dropout |
| **Slow convergence** | Poor initialization | Use Xavier/He initialization |
| **Memory error** | Batch size too large | Reduce batch size by half |

### Performance Issues

| Symptom | Diagnosis | Action |
|---------|-----------|--------|
| **R² < 0.5** | Poor feature quality | Feature engineering, more data |
| **AUC < 0.7** | Class imbalance | Use class weights, focal loss |
| **High variance** | Insufficient regularization | Increase dropout, weight decay |
| **Poor generalization** | Overfitting | Early stopping, more validation data |

### Data Quality Checks

```python
# Pre-training diagnostics
def diagnose_data(X, y):
    print(f"Shape: {X.shape}")
    print(f"Missing values: {X.isnull().sum().sum()}")
    print(f"Infinite values: {np.isinf(X.values).sum()}")
    print(f"Target range: [{y.min():.3f}, {y.max():.3f}]")
    print(f"Target std: {y.std():.3f}")
    
    # Check for constant features
    constant_cols = X.columns[X.std() == 0]
    if len(constant_cols) > 0:
        print(f"Constant features: {list(constant_cols)}")
    
    # Check for high correlation
    corr_matrix = X.corr().abs()
    high_corr = (corr_matrix > 0.95) & (corr_matrix < 1.0)
    if high_corr.any().any():
        print("High correlation detected - consider feature selection")
```

### Custom Loss Function

```python
import torch.nn as nn

class ManufacturingLoss(nn.Module):
    def __init__(self, tolerance=1.0, penalty_factor=2.0):
        super().__init__()
        self.tolerance = tolerance
        self.penalty_factor = penalty_factor
        
    def forward(self, pred, target):
        error = torch.abs(pred - target)
        
        # Standard MSE for errors within tolerance
        within_spec = error <= self.tolerance
        mse_loss = torch.mean((pred - target) ** 2)
        
        # Additional penalty for out-of-spec predictions
        penalty = torch.mean(
            torch.where(within_spec, 0.0, error * self.penalty_factor)
        )
        
        return mse_loss + penalty
```

### Batch Prediction

```python
def batch_predict(model_path, input_data_csv, output_path):
    """Process multiple predictions efficiently."""
    import pandas as pd
    
    # Load model
    pipeline = DeepLearningPipeline.load(Path(model_path))
    
    # Load data
    df = pd.read_csv(input_data_csv)
    
    # Make predictions
    predictions = pipeline.predict(df)
    
    # Save results
    df['prediction'] = predictions
    df.to_csv(output_path, index=False)
    
    return len(predictions)
```
